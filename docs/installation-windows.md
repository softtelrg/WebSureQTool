# Windows Installation Guide
This guide explains how to install **WebSureQTool** on a Windows machine.

---

# 1. Download the Installer
Open the WebSureQTool releases page:
https://github.com/softtelrg/WebSureQTool/releases/latest
Download the latest Windows installer.
Example:
WebSureQTool-1.0.0.msi

---

# 2. Run the Installer
1. Locate the downloaded file.
2. Double-click the installer.
WebSureQTool-<version>.msi
3. Follow the installation wizard.
4. Complete the installation process.
---

---

# 4. Launch WebSureQTool
After installation you can start the application from:
Start Menu → WebSureQTool
or from the desktop shortcut if one was created.

---

# 5. First Launch – Workspace Setup
When WebSureQTool starts for the first time, you will be prompted to create or select a **workspace directory**.
Example workspace location:
C:\Users<YourUser>\Documents\WebSureQTool\workspace
The workspace stores:
• Projects  
• Logs  
• Reports  
• Browser Drivers  
• Supporting tools  

---

# 6. Verify Installation
After the application launches successfully confirm:

✔ WebSureQTool starts without errors  
✔ Workspace folder is selected  
✔ Project Explorer panel is visible  
If everything loads correctly, the installation was successful.
